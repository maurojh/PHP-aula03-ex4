1 Expressões condicionais

Expressões condicionais são usadas para executar diferentes ações baseadas em diferentes condições.

Expressões condicionais em PHP:

if
if..else
if..elseif..else
switch

1.1 Comando if

O comando if executa um código se a condição for true: 

if( condição ) {
   // código executado
}

O código abaixo mostra "Bom dia!" se a hora for menor que 12: 

$hora = date("H");

if( $hora < "12" ) {
   echo "Bom dia!";
}

1.2 Exercício 1

Crie uma página que recebe o nome da pessoa e exibe: 

Bom dia, Nome!

Se a hora for menor que 12. 

Boa tarde, Nome!

Se a hora for maior ou igual à 12 e menor que 18. 

Boa noite, Nome!

Se a hora for maior ou igual à 18 e menor que 24.

1.3 Comando if..else

O comando if...else executa um código se uma condição for true e executa outro código se a condição for false. 

if( condição ) {
   // código executado se for true
} else {
   // código executado se for false
}

O código abaixo exibe Número par se o número recebido for par e Número ímpar caso contrário. 

$numero = $_GET['n'];

if( $numero % 2 == 0 ) {
   echo "Número par!";
} else {
   echo "Número ímpar!";
}

1.4 Exercício 2

Crie uma página que ajuda na cobrança de um estacionamento. A página recebe a hora de entrada e a hora de saída de um veículo. Se o veículo ficou até 2 horas no estacionamento o preço será de R$ 5,00. Caso contrário será de R$ 5,00 mais R$ 2,00 reais por hora excedida.

O sistema receberá somente a hora e não os minutos, utilize um campo de entrada com type="number".

1.5 Comando if..elseif..else

O comando if..elseif..else executa códigos diferentes para mais de uma condição. 

if( condição1 ) {
   // código executado se condição1 for true.
} elseif( condição2 ) {
   // código executado se condição2 for true.
} else {
   // código executado se nenhuma condição for true.
}

O código abaixo mostra mensagens ao longo do dia: 

$hora = date('H');

if( $hora < "12" ) {
   echo "Bom dia!";
} elseif( $hora < "18" ) {
   echo "Boa tarde!";
} else {
   echo "Boa noite!";
}

1.7 Comando switch

O comando switch seleciona um de vários blocos de código. 

switch( expressao ) {
   case rotulo1: 
      // código executado se expressao==rótulo1
      break;
   case rótulo2:
      // código executado se expressao==rótulo2
      break;
   default:
      // código executado se nenhum rótulo for encontrado
}

Como funciona:

Primeiro a expressão é avaliada e produz um valor.

O valor da expressão é comparado com cada valor de rótulo. Se for igual o comando depois do rótulo é executado até parar em break.

Se nenhum valor de rótulo for igual, o comando em default é executado.

O comando break evita que outros códigos sejam executados.
